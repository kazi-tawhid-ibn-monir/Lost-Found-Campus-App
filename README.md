# 📱 Lost & Found Campus App

A Flutter-based mobile application that helps students and staff report, search, and recover lost items within a campus environment.

---

## 📌 Description

The **Lost & Found Campus App** is designed to create a centralized digital platform where users can:

* Report lost items
* Post found items
* Search for items easily
* Connect with item owners

The goal is to simplify the recovery process and reduce the loss of personal belongings on campus.

---

## 🚀 Features

### 🔐 Authentication

* User registration & login
* Secure authentication (Firebase)

### 📦 Item Management

* Post **Lost Items**
* Post **Found Items**
* Upload images
* Add item details:

  * Title
  * Description
  * Location
  * Date & time
